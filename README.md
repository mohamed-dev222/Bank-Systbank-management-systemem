# 🏦 Bank Management System

[![C++](https://img.shields.io/badge/C++-17-blue.svg)](https://isocpp.org/)
[![Platform](https://img.shields.io/badge/Platform-Windows-lightgrey.svg)](https://www.microsoft.com/windows)

A comprehensive console-based banking system built with C++ demonstrating Object-Oriented Programming (OOP), file handling, and professional project organization.

---

## 📋 Features

### 👤 User Management
- Add, update, delete, and search for users
- Role-based permissions system
- Secure login mechanism

### 💰 Client Management
- Add new clients with account numbers
- Update client information
- Delete clients from the system
- Search for clients by account number

### 💵 Financial Transactions
- Deposit money into accounts
- Withdraw money with balance validation
- View all client balances
- Calculate total bank balances

### 🔐 Security
- Permission-based access control
- User authentication
- Password protection

---
## 🏗️ Project Structure

- Core/ → Utility classes and base logic
- Models/ → Data models (User, Client)
- Screens/ → UI screens logic
- Data/ → Stored text files (database)
- Ui/ → Main interface screens
